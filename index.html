Starting a Python TDD project

Setting up the enviroment

  1. Create a diretory and move into 
    
                            mkdir newproject 
                            cd newproject
  2. Create a virtual env and activate it 
    
                            virtualenv -p python3 venv
                            source venv/bin/activate
  3. Install dependencies 
    
                            pip install -r requirements.txt

TDD Methodology

  1. Write a test case
  2. Execute and _FAIL_
  3. Refactor till _SUCCESS_
  4. Goto 1

The problem

Write a string character compressor

  1. "a" returns "a"
  2. "aaaabbcc" returns "a3b2c2"
  3. "bbbbaabbccccc" returns "b4a2b2c5"

The rules, the funny part

  1. First statement must be RETURN
  2. Can't use any "list" method as reduce or map
  3. Can't asign variables
